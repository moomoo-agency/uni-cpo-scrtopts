This add-on makes it possible: 1) hide Text Input, but still add value to it and be able to display in cart/order meta;
the value equals to the content of connected Dynamic Notice;
2) disable Text Input, but still add value to it and be able to display in cart/order meta; the connected
the value equals to the content of connected Dynamic Notice, which also will remain hidden;
